# Safe News - Brand Style Guide

## Logo
- **Primary Logo**: See `public/logo.svg`
- **Colors**: Gradient from Blue (#0284c7) to Purple (#a855f7)
- **Usage**: Minimum 40px width for web, 64px for print

## Color Palette
- **Primary Blue**: #0284c7
- **Secondary Purple**: #a855f7
- **Accent Green**: #16a34a
- **Neutral Dark**: #1f2937
- **Neutral Light**: #f9fafb

## Typography
- **Display Font**: Poppins (Bold for headers)
- **Body Font**: Inter (Regular for body text)
- **Font Sizes**:
  - H1: 3rem (48px)
  - H2: 2.25rem (36px)
  - H3: 1.875rem (30px)
  - Body: 1rem (16px)

## Dark Mode
- **Background**: #111827
- **Surface**: #1f2937
- **Text**: #f9fafb
